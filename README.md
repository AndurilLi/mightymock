# mightymock
A mock server implemented in python, based on web.py
##Installation
```
>>python setup.py install
```
##Create a project
```
>>mocksetup mockproject 
```
##Launch 
```
>>mockstart
>>mockstart -d E:\code\MightyMock\mightymock\Sample -c default.cfg
```
